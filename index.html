Key HTML Tags

HTML tags are used to give meaning to content. Here, we’ll review some of the key tags.

Headings

HTML headings range from <h1> (main title) to <h6> (fifth-level subtitle). By default, browsers render these headings in decreasing font sizes. This can be customized with CSS.

<h1> Main Title </h1>

<h2> First Subtitle </h2>

<h3> Second Subtitle </h3>

<h4> Third Subtitle </h4>

<h5> Fourth Subtitle </h5>
<h6> Fifth Subtitle </h6>

Note: The visual size of headings is controlled by the browser. CSS can be used to modify their size, color, spacing, and more.

Paragraphs and Line Breaks

The <p> tag defines a paragraph and automatically includes a line break after its content.

Text Formatting

The following tags allow you to style text:

  • <i>: Italic text
  • <b>: Bold text
  • <em>: Emphasized text (semantic italics)
  • <strong>: Strongly emphasized text (semantic bold)
  • <mark>: Highlighted text

Lists

HTML supports two primary types of lists:

Unordered List (<ul>)

  • Item 1
  • Item 2
  • Item 3

Ordered List (<ol>)

  1. Item 1
  2. Item 2
  3. Item 3

Example: Use an ordered list for instructions (where order matters) and an unordered list for general items (where order does not matter).

Links

Links are created using the <a> tag. Examples:

Google (opens in the same tab)

Google (opens in a new tab)

Other link types:

  • <a href="http://example.com">: Link to another website.
  • <a href="page.html">: Link to an internal page.
  • <a href="#section">: Link to a section within the same page.
  • <a href="mailto:someone@example.com">: Link to open the user’s email client.
